# Raycasting engine level designer
Graphical world creator for my [raycasting engine](https://github.com/stalhein/raycaster). Allows choosing textures for the ceiling, walls, and floor, including lighting. Exports to a .wld file which can be opened by the raycaster.

It uses constraints to disallow changes such as removing walls, or covering over the spawn tile to reduce errors in the raycaster.

Very much a work-in-progress, so changes will be made, and the programme will be given additional features.

Try it at https://stalhein.github.io/raycaster_world_creator.
